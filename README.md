## LLM programs

Simple, modular components (e.g., reflection, self-consistency, tree-of-thought) for building LLM programs.

To run notebooks 01, 02, and 03, execute

```
uv venv 
source .venv/bin/activate
uv sync
```

For notebook 04-tree-of-thoughts, please use the environment within `04-tree-of-thoughts`.

```
cd 04-tree-of-thoughts
uv venv
source .venv/bin/activate
uv sync
```
